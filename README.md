# JavaScriptの登録プロセス

## 実行環境
- Visual Studio Code
- Node.js(LTS版)

## 実行手段
1. Node.js公式サイトからLTS版をダウンロードする
2. インストーラーを使用してセットアップを行う
3. ターミナルで **node -v** を実行し、インストールを確認する
4. VSCodeでプロジェクトを開く
5. ターミナルでJavaScriptファイルを実行する
